# HRSA Organ Donation Analysis

This code supports a project that analyzes characteristics affecting an individual's willingness to support organ donation.

## Statistical Methods
- Logistic regression
- Descriptive statistics

## Data Source
[HRSA 2025 Organ Donation Survey](https://data.hrsa.gov/topics/health-systems/organ_donation_opinion_survey-data#2025NationalSurveyData)
